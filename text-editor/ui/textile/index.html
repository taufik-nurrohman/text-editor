Textile text editor plugin.

Start

Include textile.min.js just after the html.min.js script:

<!DOCTYPE html>
<html dir="ltr">
  <head>
    <meta charset="utf-8">
    <title>Test Editor</title>
    <link href="font-awesome/font-awesome.min.css" rel="stylesheet">
    <link href="text-editor/ui/ui.min.css" rel="stylesheet">
  </head>
  <body>
    <p><textarea id="editor"></textarea></p>
    <script src="text-editor.min.js"></script>
    <script src="text-editor/ui/ui.min.js"></script>
    <script src="text-editor/ui/html/html.min.js"></script>
    <script src="text-editor/ui/textile/textile.min.js"></script>
    <script>
    var editor = TE.ui.Textile(document.querySelector('#editor'));
    </script>
  </body>
</html>

Configure

Instance

TE.ui.Textile(node, config);
  • node → The text area element.
  • config → Custom configuration data to override the default configuration data.
var editor = TE.ui.Textile(document.querySelector('#editor'), { … });

Configuration

The available configuration data are:

  • Inherits TE.ui() → read more
  • Inherits TE.ui.HTML() → read more
  • Inherits TE.ui.Textile()
    • states → object
      • a → object · List of default links URL for referenced–link style.

Keyboard Shortcuts

  • ⌘+b → Bold
  • ⌘+g → Image
  • ⌘+h → Header
  • ⌘+i → Italic
  • ⌘+k → Code
  • ⌘+l → Link
  • ⌘+q → Quote
  • ⌘+r → Horizontal Rule
  • ⌘+t → Table
  • ⌘++ → List
  • ⌘+- → Bullet
  • ⌘+✘ → Strike
  • ⌘+↓ → Footnote
  • ⌘+↑ → Footnote
  • ⌘+↵ → Paragraph
  • ⌘+⇧+? → Abbreviation
  • ⇥ → Indent & Depends
  • ⇧+⇥ → Outdent
  • ↵ → Depends
  • ⌫ → Depends

Depends → Special Cases

* foo|
